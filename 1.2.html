Responsive Web Design

To implement responsive web design, W3 Schools (n.d.) suggest adding a “< meta >”
tag to all web page which gives the browser “instructions on how to control the page’s
dimensions and scaling”. The name attribute can be set as viewport and the content attribute
can be set as “width=device-width, initial scale = 1.0” which allows the web page to adjust
according to the user’s device size. Responsive web design can also be applied to images by
using the style attribute and setting it to “max-wdith:100%; height:auto;” which allows the
image to scale down if needed but not to scale more than its original size.
